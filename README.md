# QcloudIMOpenSDK

## Requirements
iOS 8.0+

## Installation

QcloudIMOpenSDK is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'QcloudIMOpenSDK'
```
